Technical Architecture:

This application is engineered following clean code principles and solid Object-Oriented Programming (OOP) paradigms:

Inheritance: Utilizes a hierarchical structure where specific asset classes inherit core attributes from the Asset base class.

Interfaces: Employs the Taxable interface to enforce strict contract-based tax calculation rules across compliant assets.

Data Integrity: Implements ArrayList for dynamic storage management, ensuring efficient data manipulation and retrieval.

Core Feature:

Multi-Asset Support: Seamlessly categorize and manage distinct investment types with specific logic for each.

Automated Calculations: Real-time evaluation of total portfolio value and cumulative tax liabilities.

Interactive Command-Line Interface (CLI): Provides a user-friendly navigation system for complex portfolio operations.

Extensible Design: Architecture allows for rapid integration of new asset classes (e.g., Real Estate, Bonds) without modifying core logic.



Built With:

Language: Java
Development Environment: Visual Studio Code
Principles: OOP, Modular Design, SOLID principles.

How to Run:

Ensure Java Development Kit (JDK) is installed and configured in your PATH.
